Lecture 89. Alerts, mdn and course notes

The Window.alert() method displays an alert dialog with the optional specified content and an OK button. We usually just type alert().

The alert dialog should be used for messages which do not require any response on the part of the user, other than the acknowledgement of the message. Dialog boxes are modal windows - they prevent the user from accessing the rest of the program's interface until the dialog box is closed. For this reason, you should not overuse any function that creates a dialog box (or modal window).

Type (or data type) is a characteristic of a value affecting what kind of data it can store â€” for example in JavaScript a Boolean only holds true/false values, whereas a String holds text strings, a Number holds numbers of any kind, and so on.

Lecture 91. Variables

A variable is a named location for storing a value. That way an unpredictable value can be accessed through a predetermined name.

Lecture 92. Variables (Rules)

Naming variables - make it easy for other coders to understand. Help others understand what kind of data is in the variable. An example is naming folders on your computer - untitled folder 1, 2, 3 etc... instead of pictures, videos.

Var's can't begin with a number but can contain them, and they can't contain spaces.

  • numbers
  • letters
  • $
  • _

No other symbols are valid in variable naming.

Lecture 94. String lengths and retrieving the number of characters

Use the length property.

var name = "Dominic";
alert(name.length);

Challenge

Expected output. You have written 182 characters, you have -42 characters left.

My very wordy solution

charactersWritten = prompt("Enter Your Message: ");
charactersWrittenLength = charactersWritten.length;
maxCharacters = 140;
charactersRemaining = maxCharacters - charactersWrittenLength;
alert("You have written " + charactersWrittenLength + ", " + "you have " + charactersRemaining + " characters left");

Instructors solution

var tweet = prompt("Compose your tweet:");
var tweetCount = tweet.length;
alert("You have written " + tweetCount + " characters, you have " + (140 - tweetCount) + " characters remaining");
